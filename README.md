# Digiwell

Digiwell is a modern coaching management platform scaffolded with Next.js App Router, JavaScript, Tailwind CSS, shadcn/ui conventions, Framer Motion, and Lucide React.

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

## Architecture

This repository currently contains project architecture only. Route folders, component domains, shared utilities, service boundaries, constants, data, config, and asset folders are prepared for future development.
